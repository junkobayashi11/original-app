# アプリケーション名
「Wanted・メンバー」

## アプリケーション概要  
サッカーをしたいけれども、人数が足りない。そんな時にメンバーを募集するアプリケーションです

## URL

## 利用方法
投稿者側： 新規登録をして「メンバー募集」にて、日時や主催場所を決めルームの作成を行います。  
利用者側： トップページにて、気になる投稿にコメントをし投稿者とコネクトを取り詳細を話し合い当日の日程を決めていただきます。

## 目指した課題解決  
自主的にスポーツをしたいが人数が足りない、また試合を組みたいけれども相手の都合で試合が組めないなど監督の問題解決をするアプリケーションを目指しました。

## 課題や今後実装したい機能
アプリケーションの課題は2点あります。<br>1点目はルームを毎回検索し投稿者とコネクトする必要がある点です。  
ログインしたユーザーの情報を編集できる機能を追加しそこにお気に入り登録した内容をリンクで表示させ検索する手間を省くように実装します。  
2点目はコメントを受信するためには毎回リロードする必要がある点です。  
コメントを非同期化しリロード無しでコメントできるアプリケーションを作成します。
